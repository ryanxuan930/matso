## 決策與陷阱
- **程式碼位置**：`ai/matso_ai/*`（可 import 為 `matso_ai.*`），非 `ai/inference/`（後者只是 README 概念名；
  真正 package root 是 `ai/matso_ai/`，pyproject workspace 已安裝）。
- **latency 用注入 clock**：AIInvocationLog.latencyMs 需牆鐘，但比照全專案「注入時鐘」慣例
  （JwtCodec/Kernel），RoleManager 收 `clock=time.perf_counter`，測試用假時鐘 → 決定性。
  latency 只進 side log（AIInvocationLog），**不進 Ledger hash**（R8 教訓）。
- **無 LoRA 熱切換的本機現實**：adapter 切換以「切換次數」建模並攤銷（分組批次）；真 LoRA swap
  綁定屬部署層。RoleConfig.adapter 為字串版本標記。

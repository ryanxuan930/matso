## 測試證據
- **驗收（暴雨 vs 晴天）**：
  - 交戰：storm optical 0.4 → p_hit(storm) = p_hit(clear) × 0.4；200 次固定 seed 命中數 clear−storm > 30（可觀測分佈差異）

  - 聚合：red_loss(storm) < (clear)（暴雨降低戰鬥效率）

## 決策與陷阱
- **Core 消費 effects 不解讀氣象學**：weather 模組已把原始值轉效果係數；Core 只依 cell 效果填 env weather_modifier。
- **映射為 Core 的 v0 解讀**：交戰直瞄用 sensor_optical（能見度主導瞄準）、間瞄用 1/artillery_dispersion；偵測依 sensor_kind（OPTICAL/IR 各自係數，RADAR/聲學 v0 不受影響）；聚合用 optical；移動用 mobility。
- **weather 非硬依賴**：WeatherClient 不可達 → CLEAR（無天氣影響），不 PAUSE session（對比 terrain DOWN→PAUSE）。stale 亦由 payload 傳遞，Core 告警但不停。
- **CellEffects 解耦**：Core 端型別（app.weather.CellEffects）獨立於 modules/weather 與 proto；client 由 weather_pb2 轉入 → 維持 core/module 隔離。

## 決策與陷阱
- **離線 codegen（ADR 005）**：grpcio-tools 而非 buf generate（air-gap）。乾淨 checkout 後、跑測試/mypy 前必須先 `uv run python ops/tools/gen_proto.py`（CI/Dockerfile 已自動化）。
- **stubs 匯入**：`from matso_sdk._generated import <name>_pb2[_grpc]`；`_generated` 不入 git、ruff/mypy 排除；`--pyi_out` 讓 message 有型別，grpc service stub 無型別（mypy 邊界 override）。
- **健康三態語意**：DTED+快取→HEALTHY；缺其一→DEGRADED（缺 DTED 仍可路徑；缺快取仍可高程/LOS）；皆缺→DOWN。**外接硬碟未掛載時服務照常啟動**（DEGRADED/DOWN），不崩潰（使用者 fallback 需求）。
- **Terrain DOWN → PAUSE（SPEC §16.3）**：HealthMonitor 連 3 次失敗（RPC 不可達或自報 DOWN）→ 觸發 `SessionController.pause_all`；DEGRADED 不算失敗；恢復 → resume_all。SessionController 為 Protocol 注入介面，真正 session 暫停於 O3+ 接上。
- **斷路器**：領域 RPC 連 5 次失敗 → OPEN 快速失敗；冷卻後 HALF_OPEN 試探。時鐘注入以利確定性測試。此為執行期基礎設施（非模擬引擎），time.monotonic 牆鐘合法。
- **eta_ticks**（延續 O2.4）：仍為 ceil(total_cost) 佔位，速度換算於 O3.4。

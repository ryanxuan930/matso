## 決策與陷阱
- **純函數 + RNG 注入**：resolve_engagement 收 DeterministicRNG（stream="adjudication"），不自建；REJECTED 不擲骰（合法性是 pre-roll 的確定性拒絕），保持 RNG 序列對「合法交戰數」的決定性。
- **P_hit 單調性靠資料**：ph_by_range_band 控制點 base_ph 非遞增 + 線性插值 → 對 range 非遞增。from_base_stats 強制 range 嚴格遞增；種子模板皆非遞增。
- **係數為乘法修正、夾在 [0,1]**：terrain/weather/suppression/posture 預設 1.0（退化 base）；O5 天氣模組填真值。>1 或 <0 的極端係數不會讓 p_hit 越界。
- **間瞄免 LOS**：indirect_fire=true 的武器合法性不查 LOS（迫砲/榴彈砲）。
- **傷害資料驅動**：damage_by_armor_class；未列裝甲類別 → 0（武器無效），不拋錯。
- **事件證據性**：ai_decision 內的 p_hit/roll/coefficients 皆為確定性值 → 入 hash 安全（非牆鐘/非決定性），供 AAR 溯源。

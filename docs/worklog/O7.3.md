## 決策與陷阱
- **匯出為 JSON**（前端）/YAML（後端 dump）：`yaml.safe_load` 讀 JSON，兩者與 loader 相容；前端無需
  yaml 函式庫即可 roundtrip。
- **relations 匯出用 declarations()**：只輸出明確宣告的（非預設 HOSTILE）配對——與「未宣告預設 HOSTILE」
  語義一致，重載後等價。
- **map 拖放佈署 / MSEL 時間軸視覺化**為 UI 進階（未做）；本卡以資料模型 + 匯出匯入 roundtrip 為核心
  （SPEC §11.2 的 roundtrip 驗收已滿足）；MSEL/victory 的完整編輯 UI 待強化。

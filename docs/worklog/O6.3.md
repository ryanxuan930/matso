## 決策與陷阱
- **引用查核＝錨點存在性**（防捏造引用）：verify(citation) 解析 `collection/path.md#ANCHOR` →
  查該 collection 是否有此 (doc_path, anchor) point。相似度閾值供檢索（search）用，非 verify 的判準。
- **HashEmbedder 非語意**：僅供驗證管線正確（同文字→同向量）。roundtrip 測試以「完全相同文字」
  查回自己；真語意檢索需 bge-m3（部署層惰性載入，air-gapped 模型檔 env 注入）。
- **verifier 在 ai、協定在 core**：QdrantCitationVerifier 結構上滿足 app.guardrails 的
  CitationVerifier；core 不 import ai，注入於 O6.5 orchestration。
- **只吃 markdown**：PDF/掃描走 SPEC_INGEST/O9；ingest CLI 略過 README/_collection.md。

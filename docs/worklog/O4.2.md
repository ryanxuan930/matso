# O4.2 地圖基座（MapLibre + 離線 tile server + hillshade + H3 hex 層）

## 目標摘要
COP 地圖基座（SPEC §13.2）：MapLibre GL 地圖元件 + 自建**離線** tile server（進 compose，
env 注入 .mbtiles 路徑，缺檔優雅降級，同 terrain DTED 慣例）+ DTED hillshade 疊加層 + H3 六角
網格層（客戶端以 h3-js 計算，開關切換）。驗收：**地圖可平移縮放、hex 層開關、離線（斷網）可用**。

## 關鍵約束
- **離線可用（air-gapped，SPEC §12/§20）**：MapLibre 樣式/字型/圖示絕不依賴外部 CDN；
  無 tile server / 無網路時地圖仍可渲染（背景 + 經緯網格 + hex 層），基礎底圖由掛載的
  tileserver 提供（真 Taiwan .mbtiles 在外接硬碟 M200，env 注入、缺檔降級）。
- HOW_TO §3.2：元件 `<script setup lang="ts">`；元件放 `platform/app/components/map/`。
- MapLibre 需 DOM/WebGL → 地圖元件 client-only。

## 決策與陷阱
- **離線樣式**：不引用 demotiles.maplibre.org 等外部 style/glyphs；樣式 JSON 內嵌，基礎底圖
  來自本地 tileserver，缺席時退化為背景 + 經緯網格 + hex（全離線可證）。
- **hex 客戶端計算**：h3-js `polygonToCells`(視野 bbox) → `cellToBoundary` → GeoJSON；
  縮放時依 zoom 選 H3 res（拉遠用粗 res 避免爆量）。

## 決策與陷阱
- **契約先行 + proto mirror**：weather.proto 鏡像 weather_payload.schema.json；JSON schema 仍為權威（Core 消費）。gen_proto 納入。
- **effects 映射為 v0 佔位**（Python 常數）：真實映射於 effects_mapping.yaml + White Cell 熱調整（記 Ledger）為 O5.2。範圍嚴格夾在 schema 合法區間。
- **wind_dir 最短角插值**：處理 360° 環繞（線性會走反向）。其餘 scalar 線性。
- **SYNTHETIC 恆 stale=false**（確定性腳本）；stale 降級屬 LIVE（O5.2）。
- **檔名唯一**：跨 module 測試檔 basename 不可重複（pytest prepend import mode 會撞）——weather 測試用 test_weather_* 前綴。
- **純函數紀律**：payload/effects/synthetic 皆純函數、frozen dataclass、無 I/O；插件層才碰 gRPC。

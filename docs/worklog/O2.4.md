## 決策與陷阱
- **成本公式唯一權威 = 契約**（紅線 4）：`base × (1 + slope_penalty×slope/45)`。**不乘 cell.mobility_cost**（避免坡度重複計）。slope 夾在 [0,45]。
- **A* admissibility**：heuristic 必須是真下界才保證最佳（property「成本 ≤ 任一替代」據此）。用 `haversine ÷ (√3×平均邊長×1.10)`（低估跳數）× `min_step_cost`（slope=0 的最小 base，低估單格）。安全係數 1.10 吸收 hex 尺寸變異。
- **確定性（紅線 1）**：heap tuple 第 3 元為單調插入序 counter → f 同分時破同分穩定；鄰接 `sorted(grid_disk-{self})` 展開。同輸入永遠同一條路徑（未來 golden 需要）。
- **h3 陷阱（HOW_TO §8）**：路徑規劃不用 `h3.grid_distance`（跨 base cell 遠距可能拋例外）——只用 `grid_disk(cell,1)` 找鄰接，距離估計用中心 haversine。
- **eta_ticks 佔位**：目前 `ceil(total_cost)`（成本≈tick）。真正速度換算（每 profile 的 m/tick、油料）於 O3.4 移動執行。
- **測試地形來源**：核心正確性用手工受控 cache（可獨立算最佳解）；夾具山峰太廣不適合差異化路徑，僅作 HexGridBuilder→get_path 整合驗證。

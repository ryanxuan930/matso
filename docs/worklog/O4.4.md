# O4.4 單位渲染 + fog of war（milsymbol、intel 三級、OFFLINE 虛影）

## 目標摘要
COP 單位渲染（SPEC §13.2/§13.3）：以 milsymbol 生成 MIL-STD-2525 符號、於 MapLibre symbol
層渲染。**Fog of war**：己方實線符號（OFFLINE → 最後回報位置 + 經過時間虛影）；敵方 contact
依情報等級 DETECTED（未知菱形）→ CLASSIFIED（類型符號）→ IDENTIFIED（完整符號），並以情報
時效調透明度（愈舊愈淡）。驗收：**500 單位 ≥30 FPS**（Playwright + FPS 量測腳本）。

## 關鍵約束
- **離線**：milsymbol 生成 canvas icon（含文字：designation/OFFLINE+經過時間都烤進 icon），
  不用 MapLibre glyphs（避免字型 CDN，air-gapped）。
- 效能：500 單位 symbol 層 ≥30 FPS（MapLibre GPU symbol；icon 依 (sidc) 去重快取，非每單位一張）。
- 接 useSessionStream（O4.3）：真資料由 STATE_DIFF（己方）+ INTEL_UPDATE（contact）餵入；
  本卡以合成單位驗染 + FPS，真串流接線隨 kernel/intel 就緒。

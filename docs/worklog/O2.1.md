## 決策與陷阱
- **路徑注入**：`MATSO_DTED_PATH` 環境變數（pydantic-settings），支援外接硬碟；缺檔錯誤訊息明確提示「外接硬碟是否掛載 / MATSO_DTED_PATH 是否指向正確位置」。
- **windowed 1×1 讀取而非整檔載入 RAM**：使用者 Mac 儲存/記憶體都小、真檔在外接硬碟——rasterio 內部 block cache + OS page cache 足以達標；不做 `ds.read(1)` 全量載入。真檔 benchmark 若不達 p99<5ms 再考慮 overview/快取策略（記錄於 realdata 測試）。
- **夾具改為測試時生成**（偏離 TASKS「入 git」原文）：make_fixture.py 是確定性產生器，conftest 於 tmp 目錄生成——避免 git 存二進位、且夾具內容有單一事實來源（程式碼）。
- **出界（bbox 外）→ OutOfBoundsError** 而非回 water：出界是呼叫方錯誤（想定 bbox 應在台灣範圍內），靜默回海面會遮蔽 bug；O2.3 LOS 取樣自己保證在界內。
- terrain 為未來獨立服務（O2.5 插件化），**不 import core 的 app.***——自帶 errors/config。
